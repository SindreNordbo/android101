--- layout: default title: Assignments --- You will create a twitter-like app that communicates with an HTTP REST API for viewing and creating messages. It is up to you whether you want to focus on GUI polish and interaction, or implement all the functionality in the API.

API

The server is at: http://ios-course.herokuapp.com

It hosts the following JSON endpoints:
URL HTTP verb Description
http://ios-course.herokuapp.com/message GET Retrieves all messages
http://ios-course.herokuapp.com/message POST Creates a message(see JSON example below)
http://ios-course.herokuapp.com/message/:id PUT Updates a message
http://ios-course.herokuapp.com/message/:id DELETE Deletes a message

POST example

{
	//Date is not required, but can be fun to manipulate.;)
	"from":"hinderberg",
	"message":"You'll Never Walk Alone!:)",
	"date":"2012-10-18T17:43:48.011Z" 
}

Retrofit

If you want to use retrofit, you will need the following gradle dependency: compile 'com.squareup.retrofit:retrofit:1.6.1'. If you want to do things manually and need a framework for JSON-parsing, we suggest using Jackson.
