TextView

A TextView is used to display text. {% highlight xml %} {% endhighlight %} We use the id of the view to get the view in e.g. an activity that needs to update the text. {% highlight java %} TextView tv = (TextView) parentView.findViewById(R.id.tvText); tv.setText("New text"); {% endhighlight %} If the text is not dynamic, you should define the text in res/values/strings.xml so that it can be internationalized.

EditText

When you need text input from the user, you can use the EditText control. {% highlight xml %} {% endhighlight %} {% highlight java %} EditText editText = (EditText) findViewById(R.id.search); editText.setOnEditorActionListener(new OnEditorActionListener() { @Override public boolean onEditorAction(TextView v, int actionId, KeyEvent event) { boolean handled = false; if (actionId == EditorInfo.IME_ACTION_SEND) { sendMessage(); handled = true; } return handled; } }); {% endhighlight %}
