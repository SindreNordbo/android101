Instansmetoder

{% highlight objective-c %} - (id)init /* Gjør initialiseringen som kreves av et objekt for å bli klart */ {% endhighlight %}

Klassemetoder

{% highlight objective-c %} + (id)alloc /* Allokerer en plass i minne til objektet */ {% endhighlight %}

Ny instans av en klasse

{% highlight objective-c %} SpotifyTrack *track = [[SpotifyTrack alloc] init]; {% endhighlight %}
  1. `[[SpotifyTrack alloc] init]`, og alle metoder som starter med `init` kan sees på som konstruktører

Dot Syntax

  1. En alternativ måte å kalle tilgangsmetoder (accessor methods, get, set)
{% highlight objective-c %} NSString *artist = track.artist; track.title = @"I'm awsome!"; track.artist.firstname = @"Superman"; {% endhighlight %} {% highlight objective-c %} NSString *artist = [track artist]; [track setTitle:@"I'm awsome!"]; [[track artist] setFirstname:@"Superman"]; {% endhighlight %}

Objective-C Message Passing

  1. I Objective-C kaller man ikke metoder på objekter, men sender en beskjed
  2. Navngir ikke selve metoden, men hver parameter
  3. Skriver metoder mer som om man snakker til objektet
{% highlight objective-c %} - (id)initWithId:(NSString *)id andTitle:(NSString *)title; - (void)setTitle:(NSString *)title; - (NSString *)title; {% endhighlight %}

Java eksempel

{% highlight java %} SpotifyTrack track = new SpotifyTrack("id", "I'm awsome!"); person.setTitle("I'm awsome!"); String title = track.getTitle(); {% endhighlight %}

Objective-C eksempel

{% highlight objective-c %} SpotifyTrack *track = [[SpotifyTrack alloc] initWithId:@"id" andTitle:@"I'm awsome!"]; [track setTitle:@"I'm awsome!"]; NSString *title = [track title]; {% endhighlight %}

Terminologi

  1. Message expression {% highlight objective-c %} [person setAge:25]; {% endhighlight %}
  2. Message {% highlight objective-c %} setAge:25 {% endhighlight %}
  3. Selector {% highlight objective-c %} setAge: {% endhighlight %}
  4. Method - Koden som vil bli kjørt basert på meldingen
