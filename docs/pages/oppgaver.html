Twitter aktig app

Vi skal lage en twitter aktig app ved å snakke med en server. Her kan man utfordre seg selv på GUI flikking, eller man kan sørge for å kunne utføre alle handlingene API'et tilbyr.

Server

Serveren ligger på http://ios-course.herokuapp.com

Rest endepunktene leverer JSON og fungerer på følgende måte:
URL HTTP verb Beskrivelse
http://ios-course.herokuapp.com/message GET Henter alle beskjeder
http://ios-course.herokuapp.com/message POST Oppretter beskjed (se eksempel på JSON i body under)
http://ios-course.herokuapp.com/message/:id PUT Oppdaterer en beskjed med angitt id
http://ios-course.herokuapp.com/message/:id DELETE Sletter en beskjed med angitt id
