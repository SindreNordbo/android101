Twitter aktig app

Vi skal lage en twitter aktig app ved å snakke med en server. Her kan man utfordre seg selv på GUI flikking, eller man kan sørge for å kunne utføre alle handlingene API'et tilbyr.

Server

Serveren ligger på http://ios-course.herokuapp.com

Rest endepunktene leverer JSON og fungerer på følgende måte:

  • http://ios-course.herokuapp.com/message (GET) <- alle beskjeder
  • http://ios-course.herokuapp.com/message (POST) <- JSON i body
  • http://ios-course.herokuapp.com/message/{id} (DELETE) <- Slette
  • http://ios-course.herokuapp.com/message/{id} (PUT) <- Oppdatere
