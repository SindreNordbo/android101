Knytte ViewController klasse til ViewController i Interface builder

  1. File new -> Objective C class.
  2. Velg UIViewController som subclass
  3. Gå til storyboard, velg ViewControlleren (se bildet) og trykk i "Custom Class". Sett dette til navnet på klassen du laget. Dette vil "linke sammen" interface builder og ViewController-klassen.

Knytte interface-elementer til underliggende ViewController

  1. Velg "Assistant editor knappen" (se bilde)
  2. Velg om nødvendig relevant ".h" fil i dropdown (se bilde), slik at du har story boardet til venstre og relevant .h-fil til høyre
  3. Marker elementet (labelen i dette filfellet). Hold inne CTRL og dra fra elementet og slipp i inne i interfacet (slik bildet viser)
